# Hand Gesture-Controlled Robot

This project is a hand gesture-controlled robot that uses a webcam and computer vision to detect hand movements in real time. The system uses the cvzone and OpenCV libraries to track hands and interpret gestures, then sends movement commands to the robot wirelessly via Bluetooth (HC-05).

The robot supports two control modes: Manual Mode, where the user controls the robot's direction (Forward, Left, Right, Stop) using two-hand gestures detected by the camera, and Auto Mode, activated by a specific finger gesture, allowing the robot to navigate autonomously using its ultrasonic sensor.

This project was built as a Mars Rover simulation, combining robotics, wireless communication, and real-time image processing using Python.
